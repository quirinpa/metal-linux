Metal linux

An attempt to port Linux to WebAssembly (baby steps).

Overview

Read stdin and write to stdout. For now, that's about it, and even that is incomplete. I've made many mistakes along the way but I really want to do this right, or at least to learn from the best. It is true that this might be overkill, but I think you might just be a little grumpy today, and I already put all this work forward, so. I'm not going to stop just because you don't like it. Feel free to fix it, thou!

  • Syscalls that are not defined issue a runtime error
  • Debug syscalls may be easily defined which have a set return value (argument or constant). These print out their names, pid, and their arguments when called.
